# Apotheke Anwendung in C++
Eine Anwendung, um Produktelager zu verwalten.

Inhalt:
* MVC
* CRUD Operationen: Produkte hinzufügen, löschen, aktualiseren
* Produkte sortieren und filtern
* Benutzer-Interaktion via Konsole
* Unit-Teste
